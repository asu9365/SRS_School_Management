========================================================================
ST. ROBERT'S HIGH SCHOOL PORTAL — PROJECT SUMMARY
========================================================================

1. OVERVIEW & PURPOSE
------------------------------------------------------------------------
This project is a modernized, full-stack school management system and portal
for St. Robert's High School (SRHS), located in Jakhalabandha, Nagaon, Assam.
It serves two primary roles:
  A. A premium, modern, responsive public-facing school website for 
     students, parents, and community members.
  B. An administrative "Counter Desk" portal that allows authorized school
     officers (Admins, Clerks, Principals, etc.) to manage student details,
     teacher registrations, and parent-teacher appointments.

The application connects a React single-page frontend to a PHP REST-style 
API backend powered by a MySQL/MariaDB database.


2. TECHNICAL STACK & TOOLS
------------------------------------------------------------------------
* FRONTEND:
  - React 18.3.1 (Single-Page Application)
  - Vite 5 (Build tool & development server)
  - React Router DOM v6 (Client-side routing)
  - Lucide React (Vector icons library)

* FRONTEND STYLING:
  - Vanilla CSS design system (src/index.css) utilizing custom CSS variables
    for color palettes (Dark Navy + Orange accents), border-radii, spacing,
    box-shadows, animations, and typography (Outfit & Inter Google Fonts).
  - Modern UI elements featuring glassmorphism cards and smooth transitions.
  - Full mobile responsiveness utilizing custom CSS classes (responsive-flex,
    responsive-sticky, tab-headers, and gallery-tabs) without heavy external
    frameworks like Tailwind CSS.

* BACKEND:
  - PHP 8.1+ (RESTful API endpoints)
  - mysqli (MySQL database extension)
  - Handles cross-origin requests (CORS headers) and outputs standardized JSON data.
  - Safe error handling: DB connection wrapped in nested try-catch blocks to prevent
    PHP 8.1+ strict exception crashes.

* DATABASE:
  - MySQL / MariaDB (via XAMPP)
  - Active Port: 3306
  - Database name: school_management
  - Database Schema (database/schema.sql) defines and seeds 4 key tables:
    - login_credentials: User accounts with password-based authentication.
    - students: Student profiles, grades, parent details, and addresses.
    - teachers: Teacher qualifications, experience, and class assignments.
    - appointments: Parent-teacher interaction schedules.

* INTEGRATED SERVERS (DEVELOPMENT):
  - Vite Dev Server: Listening on http://localhost:5173/
  - Apache Web Server (XAMPP): Listening on http://localhost:8080/
  - Vite Proxy: Proxies all frontend requests starting with `/api` to the backend
    Apache address `http://localhost:8080/SRS_Project/api/`.

4. CORE FUNCTIONALITIES & PAGES
------------------------------------------------------------------------
* NAVIGATION & HEADER:
  - Sticky header bar that changes opacity on scroll.
  - Dynamic Top Utility Bar containing social media anchors and school details,
    hidden automatically on mobile viewports.
  - Accessible mobile slide-out menu drawer that handles click navigation.

* PUBLIC SCHOOL WEBSITE:
  - Home: Hero banner with CTA buttons, floating cards for key metrics, stats
    counter (25+ Years, 14 Staff, 10 Grades, 14 Computers), facility layout,
    faculty showcases, student testimonials, and a footer banner.
  - About: Rich historical timeline, interactive "Glance" cards, structural
    facility list, and a flex-stacked physical location card.
  - Gallery: Categorized view filters (All, Campus, Classes, Programs, Activities)
    that wrap fluidly, plus an overlays lightbox modal with max-image viewing.
  - Staff: Faculty profiles containing credentials, qualification badges,
    experience tags, and personal quote blocks.
  - Office Hours: Detailed schedules and an interactive appointment booking
    modal mapping directly to the MySQL database.

* ADMINISTRATION PLATFORM (COUNTER DESK):
  - Login: Authentication system using credentials stored in MySQL, providing
    role-based session tokens in LocalStorage.
  - Dashboard: Real-time statistic counters, search filters, and tabular displays
    for Students, Teachers, and Appointments, plus a detailed modal overlay.
    Responsive switcher tabs stack vertically with highlights on small screens.
  - Forms: Student/Teacher registers featuring clean columns and floating
    interactive side-previews that automatically adapt on mobile viewports.
